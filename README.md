# 📷 ESP32-CAM Smart Surveillance System

## 📌 Overview
This project implements a smart surveillance system using the ESP32-CAM module, capable of capturing images and streaming video over a network. It demonstrates the integration of IoT, embedded systems, and real-time monitoring.

---

## 🎯 Objective
- To develop a low-cost wireless surveillance system  
- To enable remote monitoring using Wi-Fi  
- To capture and stream live video feed  

---

## ⚙️ Technologies Used
- ESP32-CAM Module  
- Arduino IDE  
- Embedded C  
- Wi-Fi Communication  
- Camera Module (OV2640)  

---

## 🚀 How It Works
1. ESP32-CAM connects to Wi-Fi network  
2. Initializes the camera module  
3. Starts a web server on the device  
4. Streams live video to a browser using IP address  
5. User can access the stream remotely  

---

## ✨ Features
- Live video streaming  
- Wireless access via IP  
- Low-cost and compact design  
- Easy deployment  

---

## 🧩 Challenges Faced
- Wi-Fi connectivity issues  
- Frame rate limitations  
- Power stability  

---

## 🛠️ Solutions Implemented
- Optimized Wi-Fi configuration  
- Adjusted resolution for better performance  
- Stable power supply setup

## ▶️ How to Run
1. Open Arduino IDE  
2. Select ESP32 board  
3. Upload code to ESP32-CAM  
4. Connect to Wi-Fi  
5. Open IP address in browser
